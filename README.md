# 🛒 Saai Shopping Website

A modern and responsive eCommerce shopping website built using HTML, CSS, and JavaScript. This project focuses on clean UI design, smooth user experience, and basic shopping functionalities.

## 🚀 Live Demo
https://saai-codes.github.io/shopping-website/

## ✨ Features
- Responsive design (mobile-friendly)
- Product listing page
- Clean and modern UI
- Easy navigation between pages
- Basic shopping layout (cart, product view)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📌 Future Improvements
- Add cart functionality
- User login system
- Admin panel for managing products
- Backend integration (Node.js / Database)
